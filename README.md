🏙️ NYC Airbnb Analytics Dashboard

📌 Project Overview
This project presents an interactive Airbnb Analytics Dashboard built using Tableau to analyze listing performance in New York City.
The goal of this analysis is to uncover pricing patterns, regional trends, and property characteristics to support data-driven decision making in the short-term rental market.

🎯 Business Objectives
This dashboard answers key analytical questions:
	•	Which regions have the highest number of listings?
	•	Which regions have the highest average ratings?
	•	How does average price vary by ZIP code?
	•	What room types dominate the market?
	•	How is price distributed across the city?

🎛️ Interactive Filters
The dashboard includes dynamic filters that allow users to explore Airbnb listings across New York City based on different dimensions.
🔹 Select Property Type
Allows users to filter listings by property category (e.g., Apartment, House, Loft), enabling comparison of pricing and availability across property types.
🔹 Select Room Type
Enables filtering by room category (Entire home/apt, Private room, Shared room) to analyze how accommodation type impacts price and reviews.
🔹 Select Year
Provides time-based filtering (derived from review/date data) to examine trends in pricing and listing activity over time.


📊 Dashboard Features
